# Calculadora em React

Projeto de uma calculadora simples desenvolvida com React, utilizando Vite como ferramenta de build.

O objetivo é praticar componentes, estados e eventos no React.


## Funcionalidades

- Realizar operações matemáticas básicas
- Interface simples e responsiva
- Atualização dinâmica dos valores


## Tecnologias utilizadas

- React
- Vite
- JavaScript
- HTML
- CSS


## Como executar

1. Clone o repositório.
2. Instale as dependências com:
   npm install
3. Inicie o projeto com:
   npm run dev
4. Acesse no navegador o endereço exibido no terminal.
